# E-commerce Analysis

# วัตถุประสงค์และปัญหา
  ต้องการสร้าง Dashboard เพื่อเป็นส่วนช่วยให้บริษัท Retail ขนาดเล็กที่ยังไม่แน่ใจว่าจะทำตลาดต่างประเทศอย่างไร โดยนำข้อมูลจากเว็บไซต์ E-commerce ของสหราชอาณาจักร (UK) ที่มีฐานลูกค้าประจำจากหลายทวีปมาเป็นตัวอย่าง และวิเคราะห์พฤติกรรมการซื้อสินค้าออนไลน์ของกลุ่มลูกค้าดังกล่าว

# Multi-pages
Multi-pages แบ่งออกเป็น 2 หน้า ดังนี้ <br>
1. Custumer Overview จะกล่าวถึงภาพรวมของข้อมูล <br>
2. Custumer Analytics จะกล่าวถึงการวิเคระห์ข้อมูลเพื่อนำไปใช้สำหรับการตัดสินใจทางธุรกิจ เช่น ปริมาณคำสั่งซื้อรวมของแต่ละภูมิภาค ความถี่ในการซื้อสินค้าของแต่ละภูมิภาค มูลค่าคำสั่งซื้อโดยเฉลี่ย Pareto Analysis
